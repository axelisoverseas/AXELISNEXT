This directory contains testimonial images for the Axelis website.

- vasundhara.png - M Vasundhara Rao

- diksha.jpeg - Diksha Babbar
- ansh.jpeg - Ansh Sonchhatra

## Image Requirements:
- Format: JPG, PNG, or JPEG
- Size: Recommended 300x300px minimum
- Aspect Ratio: Square (1:1) preferred
- Quality: High resolution for crisp display
- File Size: Optimized for web (under 500KB each)

## Usage:
Images are imported and used in the TestimonialCard and TestimonialCarousel components.
